Learning JavaScript

In my Full Stack Applications course, we worked on a lab building a browser game using JavaScript. The learning goal was to use modern JavaScript in order to build a browser app with an object-oriented architecture. We ended up creating a 2D platformer using JavaScript & Canvas API. I love playing and creating platformers, so you could say I loved this lab. I loved it so much I re-designed and added some levels and also included my own assets. I included an image below of what it looks like. It is not the prettiest of assets, but I enjoyed making it my own. If you would like to check out this app, go to my Projects page!
