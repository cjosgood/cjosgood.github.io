REST Client App Using Chuck Norris API

This application is a recent homework I was given in my web applications course. The purpose for this homework was to design our own REST Client App that uses some REST API using the module design pattern. Overall, I kept this homework pretty simple as I had a bunch of other things going on. The application generates random facts from the API and displays them on the screen. When you click the new fact button, the page refreshes with a new fact. Nothing special, but I highly suggest going through a few of them because they will give you a good laugh! If you would like to check it out, go to my Projects page!
