Creating a JavaScript Browser Game

My browser game is the well-known, but classic 2D Breakout. Since this was my first ever JavaScript application, I needed a little help from other resources, so I would just like to say not all of this code is mine! I gathered from various resources, like Google, Youtube, etc. in order to put together a very simple looking game. To add my own twist to it, every time a ball hits a brick the speed of the ball gets progressively faster. I have learned how to use the canvas element to draw objects onto the screen. This was a great learning experience to work hands on with a project in order to learn a little more how JavaScript works with the browser. Time to keep on coding and if you would like to check it out, go to my Projects tab!
